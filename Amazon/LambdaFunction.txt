Lambda - сервис позволяющий запускать программный код без ваших таковых 
работающих серверов. Ваш код запускается и бежит на компьютерах "эластичной мощности" 
то есть сервис сам делает AutoScaling в зависимости от нагрузки.
Lambda понимает код на Node.js, Java, C#, Python
Запускать функции Lambda можно с помощью:
AWS Management Console
AWS CLI
API Gateway
Events: S3 bucket data change, DynamoDB update 


import boto3, os

def lambda_handler(event, context):
    myS3 = boto3.client('s3')
    try:
        results = myS3.list_buckets()
        print(results)
        output = ""
        for bucket in results['Buckets']:
            output = output + bucket['Name'] + "   "
        return ("S3 Bucket List:" + output)
    except:
        return ("ERROR!!!")
